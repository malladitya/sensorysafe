# Rivo REBRANDING GUIDE
# Complete instructions to rename Rivo to Rivo

## STEP 1: RENAME FILES MANUALLY

In File Explorer, rename these files:
1. Rivo.html → Rivo.html
2. Rivoazuremap.html → Rivoazuremap.html

## STEP 2: FIND AND REPLACE IN ALL FILES

Use your code editor's "Find in Files" feature (Ctrl+Shift+F in VS Code):

### Replace "Rivo" with "Rivo":
Find: Rivo
Replace with: Rivo
Files: *.html, *.js, *.md, *.txt, *.json

### Replace "Rivo" with "Rivo":
Find: Rivo
Replace with: Rivo
Files: *.html, *.js, *.css, *.md, *.txt

### Replace "--Rivo-" with "--Rivo-" (CSS variables):
Find: --Rivo-
Replace with: --Rivo-
Files: *.css, *.html

## STEP 3: UPDATE SPECIFIC FILES

### package.json
Change:
- "name": "sensorysafe-backend" → "name": "Rivo-backend"

### README.md
Change all instances of:
- Rivo Navigation → Rivo
- Rivo → Rivo

### server.js
No changes needed (no Rivo references)

## STEP 4: UPDATE LINKS

### In index.html:
Change: href="Rivo.html" → href="Rivo.html"

### In script.js:
Change: window.location.href = 'Rivo.html' → window.location.href = 'Rivo.html'

## STEP 5: VERIFY CHANGES

Check these files have been updated:
✓ index.html - Title, headings, links
✓ Rivo.html (renamed from Rivo.html)
✓ Rivoazuremap.html (renamed from Rivoazuremap.html)
✓ components/header.html - Logo, navigation
✓ components/footer.html - Copyright
✓ style.css - CSS variables (--Rivo- to --Rivo-)
✓ layout.css - CSS variables
✓ script.js - Comments, window.location
✓ TECHNICAL_DOCUMENTATION.md - All references
✓ DEMO_PRESENTATION_GUIDE.md - All references
✓ AZURE_AI_INTEGRATION_GUIDE.txt - All references
✓ package.json - Project name

## QUICK COMMAND (if using VS Code):

1. Press Ctrl+Shift+H (Find and Replace in Files)
2. Find: Rivo
   Replace: Rivo
   Click "Replace All"

3. Find: Rivo
   Replace: Rivo
   Click "Replace All"

4. Find: --Rivo-
   Replace: --Rivo-
   Click "Replace All"

## DONE!

Your app is now rebranded to Rivo!
